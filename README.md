# Python-Contact-Book

A command-line based Python contact book application that allows users to store, search, and manage contacts using a JSON file.  
This project demonstrates Python file handling, logging, and proper error handling.

---

## Features
- Add new contacts with name and phone number
- View all saved contacts
- Search contacts by name
- Store contacts persistently using a JSON file
- Logging for important actions and errors
- Proper error handling for invalid inputs

---

## How to Run

```bash
# Clone the repository
git clone https://github.com/samvel-32/Python-Contact-Book.git

# Change into the project directory
cd Python-Contact-Book

# Run the contact book application
python contact_book.py```
